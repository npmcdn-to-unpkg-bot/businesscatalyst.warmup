Sidebars are moved into the correct position, via javascript, no matter where they are added in the page. A Sidebar can be added to the page using a classification type to distinguish sidebars per page. The Sidebar placement is set via Javascript.
The Sidebar code structure is based on the surrounding container... <div class="sidebar-container"> Sidebar Content </div> ...and is added to container which can be adjusted as needed in the javascript.js file. <div class="sidebar col-md-3"> <div class="sidebar-container"> Sidebar Content </div> </div>
